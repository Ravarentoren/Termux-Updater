# Contributing to Termux-Updater Pro

Thank you for considering contributing!

## How to Contribute

### 1. Fork the repository
### 2. Create a feature branch
### 3. Make your changes
### 4. Submit a Pull Request to `main`

## Development Guidelines
- Follow PEP8 for Python.
- Keep functions modular and clean.
- Always test using:
  aktualizator --dry-run --verbose

## Reporting Issues
Please include:
- Termux-Updater version
- Termux version
- Android version
- Steps to reproduce
