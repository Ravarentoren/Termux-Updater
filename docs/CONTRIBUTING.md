# Contributing to Termux-Updater

We welcome contributions from developers, testers, and maintainers
of the Termux ecosystem, Android, Linux, and GitHub Actions environments.

## Collaboration guidelines
- Fork this repository.
- Create a feature branch (`feature/<your-feature>`).
- Follow the MIT + Manifest licensing policy.
- Always test in Termux before submitting PRs.
- Document any platform-specific logic clearly.

Thank you for contributing to Termux-Updater!
